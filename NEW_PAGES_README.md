# 新規ページ実装ガイド

## 概要

スクリーンショットを参考に、ラブドール関連の充実したコンテンツページを3つ作成しました。

### 作成したページ

1. **用途別おすすめ** (`page-purpose-based.php`)
   - 軽量モデル
   - リアル系ハイエンド
   - 小型モデル
   - 予算別おすすめ

2. **キャンペーン情報** (`page-campaigns.php`)
   - 現在開催中のキャンペーン
   - 定期開催のキャンペーン
   - 今後のキャンペーン予定
   - お得に購入するコツ

3. **販売店比較** (`page-shop-comparison.php`)
   - 主要販売店の比較表
   - 各ショップの詳細レビュー
   - 購入時の注意点
   - よくある質問

## インストール手順

### 方法1: 自動作成スクリプトを使用（推奨）

1. すべてのファイルをテーマディレクトリにアップロード
2. WordPress 管理画面にログイン
3. ブラウザで以下のURLにアクセス：
   ```
   https://freya-era.com/wp-content/themes/lovedoll-premium/create-pages.php
   ```
4. ページが自動的に作成されます

### 方法2: 手動で作成

1. WordPress 管理画面にログイン
2. **固定ページ > 新規追加**
3. 以下の設定でページを作成：

- **タイトル**: 用途別おすすめ

- **テンプレート**: 用途別おすすめ
- **公開**: 公開

#### キャンペーン情報ページ
- **タイトル**: キャンペーン情報
- **スラッグ**: campaigns
- **テンプレート**: キャンペーン情報
- **公開**: 公開

#### 販売店比較ページ
- **タイトル**: 販売店比較

## トラブルシューティング

### ページが表示されない

1. **パーマリンク設定を更新**
   - WordPress 管理画面 > **設定 > パーマリンク**
   - 「変更を保存」をクリック

2. **テンプレートファイルの確認**
   - テーマディレクトリに `page-*.php` ファイルが存在するか確認
   - ファイルのパーミッションを確認（644 推奨）

### スタイルが適用されない

1. **キャッシュをクリア**
   - ブラウザのキャッシュをクリア
   - WordPress のキャッシュプラグインをクリア

2. **CSS ファイルの確認**
   - `style.css` が正しくアップロードされているか確認
   - ファイルの末尾に新しいスタイルが追加されているか確認

### テンプレートが選択できない

1. **テーマを再有効化**
   - WordPress 管理画面 > **外観 > テーマ**
   - 別のテーマに切り替えてから、再度元のテーマに戻す

2. **テンプレートヘッダーの確認**
   - 各 `page-*.php` ファイルの先頭に以下が記載されているか確認：
   ```php
   <?php
   /**
    * Template Name: テンプレート名
    */
   ```
